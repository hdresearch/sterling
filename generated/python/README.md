```bash
pip install petstore_sdk
```

## Quick Start

```python
from petstore_sdk import PetStoreClient

client = PetStoreClient(
    base_url="https://petstore.swagger.io/v2",
    api_key="your-api-key",
)

# List resources
response = client.list_pets()
print(response.json())

# Create a resource
response = client.create_pet(body={"name": "Fido", "tag": "dog"})
print(response.json())

# Close the client when done
client.close()
```

## Context Manager

```python
with PetStoreClient(api_key="your-api-key") as client:
    response = client.list_pets()
    print(response.json())
```

## Async Support

```python
import asyncio
from petstore_sdk import AsyncPetStoreClient

async def main():
    async with AsyncPetStoreClient(api_key="your-api-key") as client:
        response = await client.list_pets()
        print(response.json())

asyncio.run(main())
```

- `list_pets()` — List all pets (`GET /pets`)

- `create_pet()` — Create a pet (`POST /pets`)
