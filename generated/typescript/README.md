# petstore-sdk

TypeScript SDK for **Pet Store API** v1.0.0.

Generated by [Sterling SDK Generator](https://github.com/hdresearch/sterling).

## Installation

```bash
npm install petstore-sdk
```

## Quick Start

```typescript
import { PetStoreClient } from 'petstore-sdk';

const client = new PetStoreClient({
  baseUrl: 'https://petstore.swagger.io/v2',
  apiKey: 'your-api-key',
});

// Example usage

// Info for a specific pet
const show_pet_by_idResult = await client.showPetById();
console.log(show_pet_by_idResult.data);


// List all pets
const list_petsResult = await client.listPets();
console.log(list_petsResult.data);


// Create a pet
const create_petResult = await client.createPet({ /* request body */ });
console.log(create_petResult.data);


```

## Configuration

```typescript
const client = new PetStoreClient({
  baseUrl: 'https://petstore.swagger.io/v2',   // API base URL
  apiKey: 'your-api-key',       // API key for authentication
  timeout: 30000,                // Request timeout in ms
  headers: {                     // Custom headers
    'X-Custom-Header': 'value',
  },
});
```

## API Methods


### `showPetById()`

Info for a specific pet

- **Method:** `GET`
- **Path:** `/pets/{petId}`


### `listPets()`

List all pets

- **Method:** `GET`
- **Path:** `/pets`


### `createPet()`

Create a pet

- **Method:** `POST`
- **Path:** `/pets`
