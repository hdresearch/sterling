# Tcl Language Resources

This repository also makes use of Tcl-inspired syntax and semantics. For reference, here are two authoritative resources on Tcl itself:

---

## [Tcl 8.6 Manual Pages (Command Reference)](https://www.tcl-lang.org/man/tcl8.6/TclCmd/contents.htm)

- **What it is:** The official **manual pages** for Tcl 8.6 commands.  
- **Contents:**  
  - A complete index of Tcl built-in commands (`expr`, `if`, `while`, `string`, `file`, etc.).  
  - Detailed syntax, options, and return values for each command.  
  - Links to C-level APIs and library functions for embedding Tcl.  
- **Usefulness:** This is the canonical reference when you need to check exact behavior, options, or edge cases for Tcl commands.

---

## [Tcl Tutorial Index (Tcler's Wiki)](https://wiki.tcl-lang.org/page/Tcl+Tutorial+Index)

- **What it is:** A community-maintained **tutorial collection** for learning Tcl.  
- **Contents:**  
  - Beginner-friendly introductions to Tcl syntax and semantics.  
  - Step-by-step guides on variables, procedures, control flow, and I/O.  
  - Links to more advanced topics such as object systems (OO), event loops, and GUI programming with Tk.  
  - Practical examples written and refined by the Tcl community.  
- **Usefulness:** Great for new users or for brushing up on Tcl fundamentals with examples and explanations.

---

These resources are useful companions when working with Tcl or Tcl-inspired environments such as **Eagle**, helping distinguish which features are inherited from Tcl and which are Eagle-specific.
